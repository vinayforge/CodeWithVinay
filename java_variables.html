Variables and Data Types Explained in Java

By Vinay - May 23, 2024 - 5 min read

In this tutorial we will learn Java Variables and Data Types from scratch and build a strong foundation in Java programming with simple explanations and practical examples

1. What is a Variable?

      A variable is a container used to store data in memory.

      Think of it as a labeled box:
                                Age Box → 20
                                Name Box → "Vinay"
      In Java:                        
      

int age = 20;
String name = "Vinay";

2. Variable Syntax 

      dataType variableName = value;
      Example:
        int marks = 95;

        Here:
          int → data type
          marks → variable name
          95 → value

3. Data Types in Java

      Java data types are divided into two categories:
      

        Data Types
        |
        |--- Primitive
        |--- non-Primitive

A. Primitive Data Types

        Java has 8 primitive data types:
Data Type Size Example
byte 1 byte byte age = 25;
short 2 bytes short salary = 30000;
int 4 bytes int marks = 90;
long 8 bytes long population = 8000000000L;
float 4 bytes float pi = 3.14f;
double 8 bytes double price = 99.99;
char 2 bytes char grade = 'A';
boolean 1 bit boolean isPassed = true;

B. Non-Primitive

        These are objects.
        Example:
            String
            Arrays
            Classes
            Interfaces

        Example:
            String name = "Vinay";  
      

public class Main {
    public static void main(String[] args) {

        // Declaring an array with fixed size
        int[] numbers = new int[5];

    }
}

4. Naming Rules for Variables

        Valid:
            int age;
            int studentAge;
            int _count;
            int $price;

        Invalid:
              int 1age;      // Cannot start with number
              int class;     // Reserved keyword
              int student age;  // Spaces not allowed

5. Example Code


public class Main {
    public static void main(String[] args) {

        String name = "Sam";
        int age = 21;
        double cgpa = 8.5;
        char grade = 'A';
        boolean isStudent = true;

        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
        System.out.println("CGPA: " + cgpa);
        System.out.println("Grade: " + grade);
        System.out.println("Student: " + isStudent);
    }
}

6. Final Output

After completing this tutorial, you will have a solid understanding of Java variables and data types.
