How to Build a Personal Portfolio Website

By Vinay - May 23, 2024 - 5 min read

In this tutorial we will build a personal portfolio website using HTML, CSS and JavaScript. This is perfect for beginners who want to show case their work online.

1. Project Structure

First, create a folder for your project and add the follwoing files:


    portfolio-website/
            |
            |--- index.html
            |--- style.css
            |--- script.js
