VSCode Shortcuts Every Beginner Should Know

By Vinay - May 23, 2024 - 5 min read

In this tutorial we learn some essential VS Code shortcuts that help you code faster and work more efficiently.

1. Ctrl + P

Quickly open any file in your project.

2. Ctrl + /

Comment or uncomment selected code.

3. Ctrl + Shift + P

Open the Command Palette.

4. Ctrl + `

Open or close the integrated terminal.
