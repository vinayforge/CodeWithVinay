HTML Roadmap for Beginners

By Vinay - May 23, 2024 - 5 min read

If you're starting with HTML, follow this roadmap step by step. Don't rush—practice each topic by building small projects..

1. HTML Basics

        Learn:
          1. What HTML is and how websites work

          2. Basic structure of an HTML document

          3. Tags, elements, and attributes

          4. Headings (<h1> to <h3>)

          5. Paragraphs (<p>)

          6. Comments (<!-- -->)
      

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>This is my first webpage.</p>
</body>
</html>

2. Text Formatting

        Learn:
          1. <strong>

          2. <em>

          3. <b>

          4. <i>

          5. <mark>

          6. <sup>

          7. <sub>

          8. <br>

          9. <hr>

3. Links and Images

        Learn:
          1. Anchor Tag <a>

          2. Image Tag <img>

          3. Image paths (relative and absolute)

      

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <a href="https://www.mysite.com">Visit mysite</a>
    <img src="image.jpg" alt="Description of image">
    <img src="images/profile.png" alt="Profile Picture">
</body>
</html>

4. Lists

        Learn:
          1. Ordered lists

          2. Unordered lists

          3. Description lists

5. Tables

        Learn:
          1. <table>

          2. <tr>

          3. <th>

          4. <td>

          5. colspan

          6. rowspan

      

<!DOCTYPE html>
<html>
<head>
    <title>HTML Table Example</title>
</head>
<body>

    <table border="1">

        <!-- Table Heading -->
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>City</th>
        </tr>

        <!-- Table Row -->
        <tr>
            <td>Vinay</td>
            <td>20</td>
            <td>Pune</td>
        </tr>

        <!-- colspan Example -->
        <tr>
            <td colspan="3">This cell spans 3 columns</td>
        </tr>

        <!-- rowspan Example -->
        <tr>
            <td rowspan="2">Java</td>
            <td>Beginner</td>
            <td>Level 1</td>
        </tr>

        <tr>
            <td>Intermediate</td>
            <td>Level 2</td>
        </tr>

    </table>

</body>
</html>

6. Forms

        Learn:
          1. <form>

          2. <input>

          3. <label>

          4. <textarea>

          5. <select>

          6. <button>

          Input Types:
              1. text
              2. email
              3. password
              5. number
              6. date
              7. radio
              8. checkbox

      

<!DOCTYPE html>
<html>
<head>
    <title>Simple Form</title>
</head>
<body>

    <form>

        <!-- Name -->
        <label for="name">Full Name:</label><br>
        <input type="text" id="name" placeholder="Enter your name"><br><br>

        <!-- Email -->
        <label for="email">Email:</label><br>
        <input type="email" id="email" placeholder="Enter your email"><br><br>

        <!-- Category -->
        <label for="course">Choose a Course:</label><br>
        <select id="course">
            <option>HTML</option>
            <option>CSS</option>
            <option>JavaScript</option>
            <option>Java</option>
        </select><br><br>

        <!-- Message -->
        <label for="message">Message:</label><br>
        <textarea id="message" rows="5" cols="30"
            placeholder="Write your message here"></textarea><br><br>

        <!-- Submit Button -->
        <button type="submit">Submit</button>

    </form>

</body>
</html>

7. Semantic HTML ( Very Important)

        Learn:
          1. <header>

          2. <nav>

          3. <main>

          4. <section>

          5. <article>

          6. <aside>

          7. <footer>

          These improve SEO and accessibility.

8. Multimedia

        Learn:
          1. <audio>

          2. <video>

          3. <iframe>

9. Final Output

After learning these topics, you will be comfortable with HTML and ready to move on to advanced HTML and CSS.
