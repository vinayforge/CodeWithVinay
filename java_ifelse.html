Java If-Else Statement Explained for Beginners

By Vinay - June 29, 2026 - 5 min read

In this tutorial we will learn Java if, else statemets from scratch and build a strong foundation in Java programming with simple explanations and practical examples

1. What is if-else statement?

     The if-else statement in Java is used to make decisions in a program.

     In real life, we make decisions every day:
                            If it is raining, take an umbrella.
                            Else, go outside normally.
                            If your marks are greater than 90, you get Grade A.
                            Else, you get another grade

     Similarly, in Java, we use if-else statements to execute different blocks 
     of code based on certain conditions.

2. Why do we need If-Else? 

      Without decision-making, a program would execute every statement one after 
      another without any logic.
    
      Imagine a login system:
                        If the password is correct → Login successful.
                        Else → Show an error message.   

      This is where if-else becomes useful.                  
      

// Basic Syntax
if (condition) {
    // Code executes if condition is true
} else {
    // Code executes if condition is false
}

      Explanation:
            "if" → Checks a condition.
            "condition" → An expression that returns either true or false.
            "{}"" → Contains the block of code to execute.
            "else" → Executes when the condition is false.           
            
            
        Flow of Execution:
                    1. Java checks the condition inside the "if".
                    2. If the condition is "true", the code inside the "if" block executes.
                    3. If the condition is "false", the code inside the else block executes.
                    4. Only one block executes.

3. Example 1

      Checking Age:
      

 public class Main {
     public static void main(String[] args) {
         int age = 20; 
         if (age >= 18) {
             System.out.println("You can vote."); 
            } else { System.out.println("You cannot vote."); 
        }
    }
}

        Output
             You can vote.


        How It Works
                "age" is 20.
                Java checks: "20 >= 18"
                This condition is "true".
                Therefore, the "if" block executes.

4. Example 2

            Even Or Odd Number: 
      

public class Main {
     public static void main(String[] args) {
         int number = 7; 
         if (number % 2 == 0) {
             System.out.println("Even Number"); 
            } else {
                 System.out.println("Odd Number"); 
            } 
        }
    }

                Output:
                    Odd Number


                Explanation:

                    The "%" operator gives the remainder.
                        "7 % 2 = 1"
                    Since the remainder is not 0, the number is odd.

6. Final Output

After completing this tutorial, you will have a solid understanding of Java if-else statements.
